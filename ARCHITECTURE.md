# المعمارية

## دورة المشروع

1. ينشئ المستخدم موجزًا.
2. يجمع الخادم المراجع والذاكرة المعتمدة.
3. الموجة الأولى: المواصفات والبحث والاستراتيجية.
4. الموجة الثانية: الذاكرة وUX وUI.
5. الموجة الثالثة: الكفاءة والجودة والنقد.
6. المنسق النهائي يدمج 9 مخرجات.
7. تحسب الدرجة من متوسط الثقة واكتمال الوكلاء.
8. يراجع المستخدم ويعتمد أو يطلب تعديلات.
9. يمكن توليد لوحة مرئية وموقع HTML للعميل.

## الحماية

- كل مشروع مربوط بـ `userId` ويُفحص قبل القراءة والتعديل والحذف.
- روابط المعاينة تستخدم token عام دون كشف معرف المستخدم.
- HTML المولد يُنزع منه JavaScript وiframe وobject وembed وmeta refresh.
- المعاينة تعمل داخل iframe sandbox.
- لا يتم اختلاق أرقام أو شهادات أو وسائل تواصل غير موجودة.

## التعافي

إذا انتهى الوكلاء 9/9 وتوقف الطلب الطويل، تحفظ المخرجات عند 88%. مسار `/finalize` يعيد الدمج فقط دون إعادة تشغيل الوكلاء. عند 95% ينتظر النظام دقيقتين قبل السماح بالاستكمال اليدوي.
