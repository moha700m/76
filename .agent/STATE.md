# حالة المنتج — نَسَق

> يحدّثه الأوتوبايلوت نهاية كل دورة. البشر لا يعدّلونه إلا للتصحيح.

## الأساسيات
- الاسم: نَسَق (NASQ AI)
- الواجهة: React 19 + Vite + Tailwind، RTL عربي
- الخلفية: AppDeploy SDK (ليست على Vercel)
- الاستضافة: GitHub + Vercel (الواجهة و `api/`)
- النشر الاجتماعي: dry-run

## الوكلاء التسعة
> أول مهمة لك: اقرأ `backend/design-agents.ts` واسرد الوكلاء التسعة هنا،
> كل واحد بسطر: اسمه، مدخله، مخرجه، وفي أي موجة يعمل.
